## **设置本机绑定SSh公钥，实现免密码登录**
```bash
$ 进入 C:\\Users\\Administrator(可能是自己命名)\\.ssh 目录
# 生成公钥 ssh-keygen 
$ ssh-keygen -t rsa -C "your_email"
#t：密钥类型，rsa是最常用的密钥类型
#C：邮箱public_key_email
```
*添加public key到管理平台*
