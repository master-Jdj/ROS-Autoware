C and C++

C代表的是过程性语言，而C++（python）则是面向对象语言。所以编程思路是不同的。

程序 = 数据 + 算法

算法既是对数据操作的方法

优点之一：
C++中的类是C++编程的核心对象。 类是一种将“数据”和“操作”集合封装在一起的对象，定义了数据类型和算法操作方式。
优点2：
泛型编程，既编程不需要考虑不同数据类型的差异，也可以进行操作
