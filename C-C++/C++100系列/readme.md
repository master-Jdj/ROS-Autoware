


# 编译方式：程序 C++.cpp 为例
```bash

#首先位于 C++1.cpp 所在文件夹下
$ cd C++100系列

#执行编译，生成可执行文件 C++1.exe
$ g++ C++1.cpp -o C++1.exe
# .\C++100系列> g++ ".\C++1.cpp" -o ".\C++1.exe"

#执行可执行文件.exe
$ .\C++1.exe
# .\C++100系列> ".\C++1.exe" 

```
